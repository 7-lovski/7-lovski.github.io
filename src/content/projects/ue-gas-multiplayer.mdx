这是一个基于 Unreal Engine 5.6 开发的第三人称多人联机项目。我在项目中主要完成了角色战斗、GAS 能力系统、UI 流程、EOS 联机流程，以及 Windows 客户端和 Linux 专用服务器打包。

</Fragment>
</LocalizedProse>

<LocalizedProse>
<Fragment slot="zh">

项目包含 Crunch 和 Phase 两名可操作角色。Crunch 侧重近战压制、攻击节奏和技能衔接；Phase 侧重远程瞄准、技能释放和命中反馈。我围绕这两名角色实现了对应的战斗能力、动画表现和 UI 反馈。

</Fragment>
<Fragment slot="en">

The project includes two playable characters: Crunch and Phase. Crunch focuses on melee pressure, attack rhythm, and skill chaining, while Phase focuses on ranged aiming, ability release, and hit feedback. I implemented the related combat abilities, animation presentation, and UI feedback around these two characters.

</Fragment>
</LocalizedProse>

- Organizing attributes, basic attacks, skill configuration, progression, and related UI around GAS.
- Implementing differentiated combat kits for Crunch and Phase.
- Adding movement, animation blending, IK foot adaptation, ranged aiming, minions, AI behaviors, object pooling, shop flow, and equipment crafting.
- Implementing the main menu, character selection, room / session search, local multiplayer, and EOS-based online flow.

<Fragment slot="en">

Both characters share the same GAS foundation, while their skill logic and combat pacing remain distinct.

## Networking and Release

<LocalizedProse>

</Fragment>
</LocalizedProse>

<LocalizedProse>
<Fragment slot="zh">

发布层面，我完成了 Windows 客户端打包和独立 LinuxServer 专用服务器打包。客户端包包含可运行的 Windows 可执行文件，服务端包包含 Linux 服务器入口与 EOS Linux 运行库。我也使用远端地址进行连接测试，验证客户端可以连接到服务端。

</Fragment>
<Fragment slot="en">

On the release side, I packaged both the Windows client and the standalone LinuxServer dedicated server. The client package includes a runnable Windows executable, while the server package includes the Linux server entry point and EOS Linux runtime library. I also tested remote-address connection to verify that the client could connect to the server.
